# 深入版作业任务书：从 CNN + LSTM 走到 ResNet18 + BiLSTM

## 一、任务定位

深入版对应本课程的结构升级与比较阶段。这里的重点不再是“证明流程能跑通”，而是进入课程真正要讲透的下一层：为什么基础版的轻量 CNN 可以继续升级为 ResNet18，为什么单向 LSTM 又可能进一步升级为 BiLSTM。

深入版重点承接课程中的三个知识模块：

1. 基础版 CNN + LSTM 在频谱图任务上能做到什么，又容易卡在哪里。
2. ResNet18 为什么不是“更深一点的 CNN”这么简单，而是残差学习的具体实现。
3. BiLSTM 为什么不只是“LSTM 的加强版”，而是利用双向上下文补足顺序理解。

## 二、建议前提

如果你准备直接进入深入版，建议至少已经具备以下基础：

1. 你已经完成基础版，或能提供等效的 CNN + LSTM 基线结果。
2. 你已经理解声音表示的基本逻辑，不再把频谱图当成“必须如此设置的黑箱输入”。
3. 你已经有第一轮可比较结果，而不是完全从零开始同时改很多变量。

## 三、本版的核心目标

完成深入版后，你应当能够做到：

1. 解释为什么基础版虽然能工作，但在本任务中可能仍然不够。
2. 解释为什么 ResNet18 和 BiLSTM 这两个升级点值得分别引入。
3. 基于实验比较，说明 ResNet18 + BiLSTM 是否真的带来了改进，以及改进体现在哪里。
4. 初步具备“做结构比较”的能力，而不是只会套用一个更复杂的网络。

## 四、建议按以下七步推进

### 第 0 步：先回头看基础版，到底哪里还不够

进入深入版前，不要急着替换模型。先写清楚：你为什么觉得有必要进入下一阶段。

可以从以下角度回看：

1. 基线模型在哪些鸟种上表现不错，在哪些鸟种上经常混淆。
2. 基线模型的问题更像是卷积特征提取不够，还是顺序信息利用不够充分。
3. 你是否已经观察到某些类别需要更稳定的高层特征，或者更完整的前后文信息来区分。

### 第 1 步：重新定义任务，不只是提高分数

明确本阶段的任务不是盲目追求更高结果，而是验证课程主线中的两个关键判断：

1. 更强的骨干网络是否能提取更稳定的时频特征。
2. 双向上下文是否真的帮助模型更好地理解鸟鸣顺序。

### 第 2 步：升级卷积部分，理解为什么是 ResNet18

这一阶段不只是把基础 CNN 换成另一个名字更响的网络，而是要说明：为什么课程主线选择 ResNet18 作为更合适的骨干网络。

建议重点说明：

1. 轻量 CNN 擅长提取局部纹理，但在更复杂的频谱图结构面前可能表达不足。
2. ResNet18 的价值不只是“层更深”，而是残差连接让深层特征学习更稳定。
3. 对本任务而言，更深的骨干网络意味着模型更有机会识别细致的谐波结构、纹理变化和局部模式组合。

### 第 3 步：升级时序部分，理解为什么是 BiLSTM

在本课程中，BiLSTM 不是“LSTM 的豪华版”，而是解释“为什么前后文都可能重要”的关键。你需要把它讲清楚，而不是只写一句“用了双向结构”。

建议重点围绕以下内容展开：

1. 单向 LSTM 只能从前往后累积信息，可能忽略后文对当前片段意义的影响。
2. BiLSTM 通过正向和反向两条支路同时读取序列，更适合处理上下文依赖明显的鸟鸣片段。
3. 双向建模并不一定总是更好，因此必须用实验结果说明它是否真正有效。

### 第 4 步：建立公平的比较方案

深入版最重要的一项能力，是做出有说服力的比较。你需要尽量避免“一边换主干，一边换时序模块，一边换训练策略”，最后谁也说不清原因。

建议明确以下内容：

1. 哪些数据处理设置保持不变。
2. 哪些评价方式保持一致。
3. 本次比较到底是在比较基础版 CNN + LSTM 与 ResNet18 + BiLSTM，还是拆开比较卷积升级和时序升级各自的作用。

### 第 5 步：观察结果，不只看有没有提升

这一阶段需要你把结果拆开看，而不是只报告一个总指标。你需要回答：ResNet18 + BiLSTM 这条升级结构到底改变了什么。

建议重点观察：

1. 总体结果是否提升。
2. 哪些类别提升明显，哪些类别变化不大。
3. 错误类型是否发生变化。
4. 训练过程是否更稳定，还是只是偶然取得更高结果。

### 第 6 步：把结构讲成一条完整的数据流

课程中的深入版模型不是零散知识点，而是一条完整流水线。你需要能把这条流水线讲顺：输入如何变化、每个模块在做什么、最终为什么能够输出类别判断。

建议至少讲清楚以下链条：

1. 原始音频如何进入频谱表示。
2. ResNet18 在频谱图上负责什么。
3. BiLSTM 在序列维度上负责什么。
4. 分类头如何把高层特征变成最终判断。

## 五、建议至少保留的产出

1. 基础版复盘与升级理由。
2. ResNet18 与 BiLSTM 的结构说明图。
3. 控制变量下的比较设计表。
4. 基础版与深入版的对比结果。
5. 对“提升来自哪里”的结构化分析。

## 六、LLM 在深入版中的更合理用法

### 建议重点用于

1. 帮你解释残差连接、双向上下文等概念。
2. 帮你一起审视比较方案是否存在变量混乱。
3. 帮你把“结果变化”翻译成“结构理解”。
4. 帮你复盘为什么某些提升不明显，避免只停留在主观判断。

### 你应当保留的判断

1. 这次改进到底想验证什么，应由你自己定义。
2. 哪些解释真正符合你的实验现象，应由你自己筛选。
3. 哪些结论只是假设、还缺证据，也应由你自己明确写出。

## 七、深入版完成标志

满足以下状态时，可以认为你已经完成深入版：

1. 你已经构建出比基础版更有教学代表性的 ResNet18 + BiLSTM 模型。
2. 你能清楚解释基础版与深入版的主要差异。
3. 你已经完成至少一次有效的模型比较，而不是只展示单个结果。
4. 你能说出深入版结构的优势、边界，以及是否值得继续进入 Transformer 等探索结构。
